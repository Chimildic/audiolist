Audiolist

Перенос, фильтрация, сортировка музыки на Android

Установить
Подробнее

Импорт

Плейлисты, любимые треки, текстовые файлы. История прослушиваний, включая Spotify с первой секунды.

Фильтры

Сложение, вычитание, смешивание пропорциями, удаление дубликатов и истории, отбор по мета данным.

Перенос

Между Spotify, Яндекс Музыкой и другими источниками с возможностью ручного уточнения результатов автопоиска.

Экспорт

Создание или обновление плейлистов, лайкер, вывод в файл, локальное накопление для будущих запусков.

Автоматизация

Запуск программ по расписанию и ярлыку. Интеграция с Tasker для запуска по местоположению, голосом и физической кнопкой.
